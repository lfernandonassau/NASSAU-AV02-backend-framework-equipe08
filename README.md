##  Sobre o Projeto

O **Sistema de Contas** é uma aplicação completa de gerenciamento de despesas, composta por:

- **Frontend em React (Vite)**
- **Backend em Laravel (API REST)**
- Login com **autenticação JWT**
- **CRUD completo** de contas
- Filtros, paginação e pesquisa
- Comunicação via **Axios**
- Interface rápida e responsiva

O objetivo é permitir que o usuário gerencie suas próprias contas de forma simples e eficiente.

---

##  Tecnologias Utilizadas

###  **Frontend**
- React + Vite  
- React Router  
- Axios  
- Bootstrap  
- Context API + LocalStorage  
- Hooks (useState, useEffect)

###  **Backend**
- Laravel 10  
- JWT ou Sanctum  
- MySQL  
- Eloquent ORM  
- Controllers + Models + Migrations
